# bubble-make

实现一个泡泡发生器的效果，点击鼠标，产生泡泡，泡泡在画面中随意漂浮，达到生命极限即可自动消失

构造函数方式定义bubble对象，其中包括bubble的属性和绘制方法  

requsetAnimationFrame实现动画效果  
同时，cancelAnimationFrame停止执行的动画
